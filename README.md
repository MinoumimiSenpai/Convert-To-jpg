• Convert-To-jpg
Batch Converts all files in "Current Folder" 
From jpeg-png-jfif-webp To jpg

• Convert-To-jpg-[Delete]
Batch Converts all files in "Current Folder" 
From jpeg-png-jfif-webp To jpg
Then Deletes converted files (except jpg)

• Convert-To-jpg-[Sub][Delete]
Batch Converts all files in "Current Folder" + "All Sub Folders"
From jpeg-png-jfif-webp To jpg
Then Deletes converted files (except jpg)

Might Require ImageMagick Installed 
https://imagemagick.org/script/download.php#windows
